# DJango + React CRUD

Este es un CRUD usando una arquitectura BackEnd y FrontEnd desacoplado

en lado BackEnd se usa: Python + Django + DRF Spectacular 

en lado FrontEnd se usa:  Bun + Vite + ReactJs + Axios + Tailwind CSS

# Comandos

Iniciar el servidor (BackEnd) con python + django

```shell
$ ./.venv/Scripts/active.bat # En Windows
$ source ./.venv/Scripts/active # En GitBash, Linux, MacOS
$ python manage.py runserver
```

Iniciar el cliente (FrontEnd) con bun + reactjs

```shell
$ cd cliente
$ bun dev
```
